# Install

## install chezomi

```bash
# mac
brwe install chezomi
# ubuntu
snap install chezmoi --classic
# Arch
pacman -S chezmoi
```

## clone dotfile

```bash
chezmoi init https://github.com/mochi-sann/dotfiles.git
```

## backup files 
```bash 
chezmoi cd 
# in chemzoi directory
chezmoi archive
``` 
## apply chezmoi

```bash
chezmoi apply
```
